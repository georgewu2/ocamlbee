The Rules

Rules of the Bee

  • Players first will hear the problem statement.
  • Players can ask about the problem (constraints, edge cases / etc.) for 1 minute.
  • Players cannot talk to one another after the problem statement is given except to declare readiness.
  • The instance a Player starts to spell a letter, time will start.
  • Players will alternate, either giving one “command” per turn.
  • Players can use the standard keyboard and the “Clearline” option which will delete the entire line that the two teams are working on.
  • For these specific keywords, use the following phrases:
    • ‘\t’ : “TAB”
    • ‘\n’ : “NEWLINE”
    • ‘clear’ : ‘CLEARLINE’
    • ‘#’ : ‘POUND’
    • ‘[‘ : ‘OPEN SQUARE BRACKET’
    • ‘]’ : ‘CLOSE SQUARE BRACKET’

Functions

  • Must be of form: def fun(....)
  • Watch out! All tabs and newlines must be correct
  • Watch out for runtime errors (divide by zero, interpretation error, etc.)

Teams

  • Each team must have 2 members
  • Sign up above!

Time Limit

  • Each team must have 2 members
  • Sign up above!

Scoring

  • Time matters...
  • ...But only if you are correct!
